## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git](https://github.com/aaqibsyed/portfolio-aj.git
```

Navigate to project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```
